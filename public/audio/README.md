# Audio Files

Add your audio files here. Reference them in `src/data/tracks.js`.

Required files:
- track-01.mp3 — first music track (piano/ambient for landing page)
- track-02.mp3 — second track (romantic instrumental)
- track-03.mp3 — third track
- voice-note.mp3 — your personal voice recording for the Voice Note Player

Supported formats: mp3, ogg, wav
